# 快速开始指南

## 项目简介

文档智能比对分析工具是一款基于 AI 技术的在线文档比对工具，可以快速比对两篇 Word 文档的差异，并使用文心大模型进行智能分析。

## 功能特点

✨ **智能比对** - 词级别的精确差异检测  
🎨 **可视化展示** - 直观的颜色高亮显示  
🤖 **AI 分析** - 文心大模型智能分析差异  
📊 **专业报告** - 结构化的分析报告  
🚀 **流式响应** - 实时显示 AI 分析结果  

## 技术栈

- **前端框架**: React 18 + TypeScript
- **UI 组件**: shadcn/ui + Tailwind CSS
- **构建工具**: Vite
- **文档解析**: mammoth
- **差异比对**: diff
- **AI 服务**: 文心大模型

## 安装步骤

### 1. 安装依赖

```bash
npm install
# 或
pnpm install
```

### 2. 配置环境变量

确保 `.env` 文件中包含以下配置：

```env
VITE_APP_ID=app-7m0ueu4u3lz5
```

### 3. 启动开发服务器

```bash
npm run dev -- --host 127.0.0.1
```

访问 `http://127.0.0.1:5173` 即可使用。

## 使用方法

### 第一步：上传文档

1. 在页面顶部找到两个上传区域
2. 左侧上传原始文档（文档1）
3. 右侧上传修改后的文档（文档2）
4. 支持拖拽或点击上传 .doc/.docx 文件

### 第二步：开始比对

点击中间的 **"开始比对分析"** 按钮，系统会自动：
- 解析文档内容
- 进行差异比对
- 调用 AI 分析

### 第三步：查看结果

比对完成后，页面会显示：
- **左栏**: 原始文档，红色标记删除内容
- **中栏**: 修改文档，黄色标记新增内容
- **右栏**: AI 智能分析报告

## 常见问题

### Q: 上传文档后没有反应？
A: 请确保上传的是有效的 .doc 或 .docx 格式文件。

### Q: AI 分析失败？
A: 请检查网络连接，确保可以访问文心大模型 API。

### Q: 文档比对结果不准确？
A: 本工具使用词级别比对，对于格式复杂的文档可能存在偏差。
